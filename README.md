## ✨ 核心功能 (v5.0 全新升级)

### 📺 智能排队刷课 (降维打击版)
* **精准识别**：穿透复杂的网页嵌套框架，直接提取底层数据，精准锁定真实的未完成任务。
* **自动执行**：视频与图文 PPT 智能排队，完成一个自动接管下一个，互不干扰。
* **视频极速静音**：默认 2.0x 倍速播放（支持手动切换），自动静音，自带防暂停机制，无视切屏或失焦限制。
* **暴力文档翻页**：光速滑到底部，破解各类隐藏滚动条的打卡限制。

### ⏭️ 无感连播与智能跳过
* **跳过已完成**：透视底层数据，已变绿的任务点绝对不重复刷。
* **测验免打扰**：遇到章节测验自动放行，绝不卡住学习进度。
* **强穿拦截弹窗**：所有任务处理完毕后自动点击“下一节”，并在 500 毫秒内自动粉碎“还有任务点未完成”的警告弹窗，强制通行。

### ⛏️ 题库抓取与自动答题
* **题目抓取**：一键遍历并抓取当前页面的所有题目。
* **多格式导出**：支持将抓取的题库一键导出为 Excel 或 Word 格式，方便日后复习。
* **本地自动答题**：支持上传自定义 Excel 题库，设定答案列后，自动匹配并填写答案。

## 🚀 使用指南

扩展程序共分为三大独立运行的模块，界面清爽，互不干扰：

1.  **📺 刷课模块**：进入带有视频或文档的章节页面，打开面板，点击“**▶ 一键完成页面任务**”。你可以去喝杯咖啡，它会自动看完当前页并跳到下一章。
2.  **⛏️ 抓取模块**：在测验页面，点击“**▶ 开始自动抓取**”，配合系统的翻页功能收集题目，完成后点击导出。
3.  **⚡ 答题模块**：进入考试页面，点击“**📂 上传题库**”导入你的 Excel 答案文件，选好答案所在的列，最后点击“**⚡ 开启自动答题**”。
