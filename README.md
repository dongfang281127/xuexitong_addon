## ✨ 核心功能

* **⛏️ 题目抓取**：自动翻页采集章节测验题目，一键导出。
* **💾 多格式导出**：支持导出 **Excel** (.xlsx) 或 **Word** (.doc) 格式。
* **⚡ 自动答题**：在考试页面，利用本地 Excel 题库自动匹配并填写答案。
* **🧠 智能匹配**：完美支持选项乱序，精准识别多选题，防止误判。
* **🖥️ 交互升级**：可拖拽、可折叠的悬浮面板，不遮挡视线。

## 📖 使用指南

### 1. 建立题库
1.  进入课程章节测验页面。
2.  点击面板上的 **“⛏️ 切换模式”** -> **“▶ 开始自动抓取”**。
3.  抓取完成后，点击 **“💾 导出题库”**。
4.  打开 Excel，在 **G列（通常是G列）** 填入正确答案并保存。

### 2. 自动答题
1.  进入考试页面。
2.  点击 **“📂 上传题库”**，选择刚才的 Excel 文件。
3.  点击 **“⚡ 开启自动答题”**。
